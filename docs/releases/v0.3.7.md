# v0.3.7 同步上游 v6.1.1，并补 GitHub About 的 ZCode 描述

这版主要是跟进 `obra/superpowers` 的 `v6.1.1`，同时把 GitHub 仓库右侧 About 描述里漏掉的 ZCode 支持补上。

## 更新了什么

- vendored 上游 `obra/superpowers` 已从 `v6.0.3` 更新到 `v6.1.1`
- README 顶部的 vendored 版本号同步更新为 `v6.1.1`
- GitHub 仓库 About 描述补上 `ZCode`
- GitHub topics 补上 `zcode`

## 上游这次主要变了什么

- `v6.1.0` 压缩了 `using-superpowers` bootstrap 和部分 harness reference，减少每个会话的固定 token 成本
- `v6.1.0` 移除了 Gemini CLI 支持，因为 Gemini CLI 已经 EOL
- `v6.1.0` 调整 Codex 插件支持，去掉 Codex SessionStart hook，改为依赖 Codex 原生 skill 触发
- `v6.1.1` 修正 Codex manifest，显式声明 `hooks: {}`，避免 Codex fallback 到 Claude 的 SessionStart hook
- `v6.1.1` 清理了已经没有调用方的 Codex session-start 文件，并补了 Codex portal package 脚本

## 已验证

- PowerShell 单测通过
- ZCode 单目标项目安装 smoke 通过，真实 `v6.1.1` 装出 14 个 skill
- `install-all.ps1 -Targets All` 项目级 smoke 通过，7 个目标都装出 14 个 skill
- 中文触发映射检查通过：`missing=0`、`extra=0`
