# v0.3.3 安装脚本修复版

这版主要是修安装脚本里的一个真实问题，顺手把提示语也讲清楚了。

直接说重点：

- 修复了 `AGENTS.md`、`CLAUDE.md`、`CODEBUDDY.md` 这类说明文件已经存在但内容为空时，安装脚本可能直接报错的问题
- `install-all.ps1`、`update-all.ps1`、`refresh-upstream-and-reinstall.ps1` 现在会先明确告诉你这次要处理哪些工具
- 不传 `-Targets` 时，也会直接说明当前走的是“全部支持工具”的默认行为，少一点误解

如果你前面刚好被空文件安装失败卡住，这版就是修这个的；如果你没遇到过，更新后主要是提示更清楚、安装过程更省心。
