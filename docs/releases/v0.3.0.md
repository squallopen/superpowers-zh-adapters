# v0.3.0 Claude Code / Codex 适配正式版

这版主要就是三件事：

- 正式支持 `Claude Code`
- 正式支持 `Codex`
- 把 vendored 上游 `obra/superpowers` 同步到 `v5.0.6`

`Codex` 这次不只是“能装上”，而是把最容易踩坑的 git 场景先限住了，所以用起来会稳很多。

重点就这些：

- 如果当前已经在 linked worktree / detached HEAD 里，就不会再盲目新建 worktree
- 如果当前环境不允许 `branch`、`push` 或建 PR，就会老实走 `commit + handoff`
- `Claude Code` 和 `Codex` 的中文适配、安装入口和说明文档也一起补齐了

如果你主要在 `Claude Code` 或 `Codex` 里用这套适配，这版就值得升。
