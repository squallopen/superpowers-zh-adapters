# v0.3.5 同步上游 v5.1.0

这版主要是跟进 `obra/superpowers` 的 `v5.1.0`。如果你之前已经装过这套中文适配，直接更新就行，不需要重新学一套命令。

直接说重点：

- 上游内容从 `v5.0.7` 更新到 `v5.1.0`
- 还是原来的 14 个 skill，中文触发词不用改
- Codex 代码审查相关说明跟着上游改了，不再指向上游已经删掉的旧 reviewer 文件
- worktree 和分支收尾的规则更稳了，尤其是已经在隔离工作区或 detached HEAD 里的情况
- OpenCode 插件现在会缓存启动提示，少做重复读文件和解析

安装脚本没有换入口。第一次安装还是用 `install-all.ps1`，已经装过就用 `update-all.ps1`，想连上游一起刷新再重装就用 `refresh-upstream-and-reinstall.ps1`。
